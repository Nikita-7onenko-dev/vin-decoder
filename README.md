# VIN Decoder SPA

Single Page Application для декодирования VIN-кодов с использованием API NHTSA.

🔗 **Live demo:**
([Открыть демо на GitHub Pages](https://nikita-7onenko-dev.github.io/vin-decoder/#/))

---

## 📌 Описание

Приложение позволяет:

* декодировать VIN-код автомобиля
* просматривать характеристики автомобиля
* видеть предупреждения и ошибки декодирования
* повторно использовать последние запросы
* изучать список всех переменных (характеристик) и их описание

---

## ⚙️ Реализованный функционал

### 🔍 VIN Decode

* Ввод VIN-кода с базовой валидацией:

  * не пустое значение
  * максимум 17 символов
  * только допустимые символы

* Декодирование VIN через API

* Отображение:

  * таблицы характеристик
  * ошибок декодирования
  * предупреждений

---

### ⚠️ Обработка ошибок

Реализовано разделение на два уровня:

#### 1. Transport errors

(обрабатываются React Query)

* network errors
* server errors

#### 2. Decode errors (бизнес-логика API)

Так как API всегда возвращает `200 OK`, ошибки извлекаются из данных ответа:

* ошибки VIN (например, invalid check digit)
* предупреждения (например, partial data)

Используется `ERROR_MAP` для преобразования кодов ошибок в понятные сообщения.

## 🧠 Особенности реализации

### 📡 Работа с API

API NHTSA имеет особенности:

* всегда возвращает `200 OK`
* поле `Message` не отражает реальные ошибки

➡️ Поэтому:

* реализована собственная логика обработки ошибок
* ошибки извлекаются из массива `Results`

---

### 📦 Кэширование

* используется React Query
* `staleTime: Infinity` для variables
* повторное использование данных без лишних запросов

---

### 🧩 Обработка HTML в описаниях

API возвращает поле `Description` в виде HTML-строки.

Для корректного отображения в интерфейсе используется библиотека `html-react-parser`, которая преобразует HTML-строку в React-элементы.

## 🚀 Запуск проекта локально

```bash

npm run dev

```
http://localhost:5173
